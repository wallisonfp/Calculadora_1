# Calculadora Linux

Projeto desenvolvido durante os estudos de Linux e Python.

## Arquivos do projeto

* calculadora.py → versão em Python
* calculadora.sh → versão em Shell Script
* comandos.txt → comandos utilizados durante a atividade

---

## Como executar o arquivo .sh

### Dar permissão de execução

```bash
chmod 744 calculadora.sh
```

### Executar o script

```bash
./calculadora.sh
```

---

## Como executar o arquivo Python

Certifique-se de possuir o Python instalado.

Execute:

```bash
python3 calculadora.py
```

---

## Explicação do código Python

O programa funciona como uma calculadora simples.

### Etapas

1. Recebe dois números digitados pelo usuário.
2. Recebe a operação matemática desejada.
3. Utiliza estruturas condicionais (`if`, `elif` e `else`) para identificar a operação.
4. Exibe o resultado na tela.
5. Trata erros de operação inválida e divisão por zero.

### Conceitos utilizados

* Variáveis
* Entrada de dados (`input`)
* Saída de dados (`print`)
* Estruturas condicionais
* Operadores matemáticos
* Conversão de tipos (`float`)
